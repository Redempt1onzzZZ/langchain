# OpenWebUI

>[OpenWebUI](https://docs.openwebui.com/)  is an `extensible`, feature-rich, and user-friendly self-hosted AI platform designed to operate entirely offline. It supports various LLM runners like Ollama and OpenAI-compatible APIs, with built-in inference engine for RAG, making it a powerful AI deployment solution.


## Installation and Setup

Install several Python packages:

```bash
pip install open-webui
````


## Chat models


See a [usage example](/docs/integrations/chat/openwebui).

```python
from langchain_community.chat_models import OpenWebUIAI
```
